# BLI Exercises

Self-paced interactive exercises for [Big Lever Institute](https://bigleverinstitute.org). Each exercise is a standalone HTML page that teaches one reusable Claude/LLM skill while producing a tangible tool the learner keeps.

Live at: <https://exercises.bigleverinstitute.org/>

## Governing documents

- **[BLI-ADR-001](./BLI-ADR-001_Exercise_Deliverable_Architecture.md)** — what every exercise is and why. The architectural contract.
- **[HOSTING.md](./HOSTING.md)** — how exercises are hosted, embedded in Circle, measured, and published. The operational playbook.

Exercise 01 (Resume Tailorer) is the canonical reference implementation for exercises 02–25.
